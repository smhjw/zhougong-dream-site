# 周公解梦网站

## 项目简介
这是一个纯前端的周公解梦在线网站，包含：
- 梦境关键词解读
- 热门关键词快捷查询
- 本地历史记录
- 抽签式每日梦境运势页

## 本地运行
1. 直接双击打开 `index.html`
2. 或在当前目录启动静态服务后访问

```powershell
python -m http.server 8080
```
访问：`http://localhost:8080`

## 文件说明
- `index.html`：主页面（关键词解梦）
- `app.js`：主页面交互逻辑和词库匹配
- `daily.html`：每日抽签页面
- `daily.js`：每日抽签逻辑（含“今日一签”本地存储）
- `styles.css`：全站样式与动画

## 接入在线解梦接口（可选，但推荐）
默认是“本地词库优先”，未命中时会走“本地兜底”。你也可以启用在线接口，让更多关键词都有反馈。

配置位置：`app.js` 中的 `ONLINE_API_CONFIG`

做法：
1. 选一个 provider（例如天行数据/极速数据）
2. 将 `enabled: false` 改成 `true`
3. 填入你在该平台申请到的 `key`

注意：
- GitHub Pages 上运行的是纯前端页面，API Key 会暴露在浏览器里，不适合放敏感 key。
- 更安全的做法是加一个你自己的后端/云函数做代理，把 key 放在服务端，再由前端调用你的接口。
