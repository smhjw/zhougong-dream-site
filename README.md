# 周公解梦网站

## 项目简介
这是一个纯前端的周公解梦在线网站，包含：
- 梦境关键词解读
- 热门关键词快捷查询
- 本地历史记录
- 抽签式每日梦境运势页

## 本地运行
1. 直接双击打开 `index.html`
2. 或在当前目录启动静态服务后访问

```powershell
python -m http.server 8080
```
访问：`http://localhost:8080`

## 文件说明
- `index.html`：主页面（关键词解梦）
- `app.js`：主页面交互逻辑和词库匹配
- `daily.html`：每日抽签页面
- `daily.js`：每日抽签逻辑（含“今日一签”本地存储）
- `styles.css`：全站样式与动画
